## Questions

### 1. Difference between var, let, const
var is function scoped. let and const are block scoped. const cannot be reassigned.

### 2. Spread Operator
The spread operator (...) expands elements of arrays or objects.

Example:
const arr=[1,2]
const newArr=[...arr,3]

### 3. map filter forEach
map returns a new array.
filter returns elements that match a condition.
forEach only loops through items.

### 4. Arrow Function
Short syntax for writing functions.

Example:
const add=(a,b)=>a+b

### 5. Template Literals
Template literals allow embedding variables using backticks.

Example:
`Hello ${name}`
